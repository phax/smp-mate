## Building

Voraussetzungen: Java 1.8 oder höher, Apache Maven 3.x

Befehl: `mvn clean install`

Das Ergebnis ist die Datei `target/smp-mate-x.y.z-SNAPSHOT-app.jar` wobei `x.y.z` für die Versionsnummer steht
